# M.League 复盘

本项目用于复盘立直麻将联赛（M.League）每日比赛，
以表格形式记录每个半庄选手的何切内容，
帮助麻将爱好者分析和学习职业选手的打法。

## 功能特性

- 以表格形式记录和展示比赛数据
- 支持选手信息管理
- 日历视图查看比赛安排
- 复盘评论和分析功能

## 技术栈

- **React** - 前端框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Shadcn UI** - UI 组件库
- **Tailwind CSS** - 样式框架
- **Lucide React** - 图标库
- **ESLint** - 代码规范

## 开发

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建生产版本
pnpm run build

# 代码检查
pnpm run lint
```
